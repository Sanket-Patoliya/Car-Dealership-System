# CarHub - Car Dealership Inventory System (Frontend Portal)

This is the React-Vite based frontend client for CarHub, styled with Tailwind CSS, utilizing React Router DOM for routing and Axios for API requests.

## Tech Stack
- React with Vite
- Tailwind CSS
- Axios
- React Router DOM

## Rebranding & Debugging Changes
- Modern Light Theme: Transformed the user interface from dark cyber themes to a clean, crisp, light-mode palette (white & slate-50).
- Fixed Authentication Role Mismatch: Enabled uppercase role verification for `'ADMIN'` users.
- Secured Route Protection: Restricts layout paths based on active authentication token and admin status tags.
